## 🚀 デプロイ手順（簡易版）

### ステップ1: ファイルをダウンロード
すべてのファイルをダウンロードして1つのフォルダに配置

### ステップ2: GitHubにアップロード
```bash
git init
git add .
git commit -m "Initial commit: 経穴ビー玉ソリティア PWA版"
git remote add origin https://github.com/あなたのユーザー名/リポジトリ名.git
git push -u origin main
```

### ステップ3: Vercelでデプロイ
1. https://vercel.com にアクセス
2. GitHubアカウントで連携
3. リポジトリを選択してインポート
4. 「Deploy」をクリック
5. 完了！

詳細は **DEPLOYMENT.md** を参照してください。
